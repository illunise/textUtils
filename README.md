
# TextUtils

Website that will perform actions like Remove Punctuations, Text All Caps, New Line Remover and Extra Space Remover from the text.
## Features
 - Remove Punctuations
 - Text All Caps
 - Remove New Line
 - Remove Extra Spaces

## Requirements
 - Python 3.10
 - Django (v4.1.5)
